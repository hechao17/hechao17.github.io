新的开始

这是新的开始，我用hexo创建了第一篇文章。

通过下面的命令，可以创建一篇新的文章。

在git bash中
hexo new 新的开始

然后启动hexo本地服务器。

hexo server

通过浏览器打开， http://localhost:4000/, 就会出现我们新写的文章。

hexo deploy 发布到github中会报错
ERROR Deployer not found: git
需要使用下面这条指令
npm install --save hexo-deployer-git
