新的开始

这是新的开始，我用hexo创建了第一篇文章。

当然是需要在电脑上安装好hexo的环境，包括：

  1. 安装git
  2. node.js
  3. hexo

这些随笔在网上找找就完成了。

然后在你需要的目录下，使用git bash窗口。通过下面的命令，可以创建一篇新的文章。

在git bash中
hexo new 新的开始

然后启动hexo本地服务器。

hexo server

通过浏览器打开， http://localhost:4000/, 就会出现我们新写的文章。

然后是部署到git hub

部署完成后，我们就可以通过网站来访问你的博客了，其中username是你的github的用户名 

https://username.github.io/

我们需要在 github 中先创建一个仓库，命名叫做 username.github.io 否则后续操作会比较省心！！

部署的时候，我们在git bash中使用指令

git g    #generate 就是生成所需要的网站
git d    #deploy 上传到github的服务器中

完成后就可以通过 网页访问你的博客啦~~~~
